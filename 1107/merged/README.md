# 미니게임 허브 (Runner + Aim Trainer)

두 프로젝트(GameProject, gamp_clicker)를 합쳐 러너와 에임 트레이너 두 게임을 하나의 페이지에서 선택해 즐길 수 있도록 통합했습니다.

## 구성
- 러너: 2D 캔버스 러닝 게임 (고급 버전: 레벨업 카드, 로켓, 실드, 라이프 포함)
- 에임 트레이너: Babylon.js 기반 3D 타겟 슈팅 (30초 제한, 콤보/정확도 통계)
- 낚시: 캔버스 기반 간단 낚시 미니게임 (캐주얼 플레이용)

## 실행 방법
- 파일을 직접 더블클릭해 여는 y식(file://)은 일부 사운드/경로 제한이 있어 비추천입니다.
- 정적 서버로 여는 것을 권장합니다. 아래 예시는 Windows cmd 기준, Python 내장 서버 사용:

```cmd
:: 프로젝트의 merged 폴더를 서버 루트로 사용해야 합니다 (Runner가 "/image/" 절대경로를 사용)
cd merged
python -m http.server 5173
:: 브라우저에서 http://localhost:5173/ 접속
```

## 단축키/조작
- 러너: 스페이스/위=점프(꾹 누르면 높이), 아래=슬라이딩/급강하, R은 사용하지 않음(좌상단 메뉴 버튼 사용)
- 에임 트레이너: 좌클릭 발사, [ / ] 로 감도 조절, 첫 클릭 시 포인터락 및 오디오 활성화

## 에셋
- 러너 이미지: `merged/image/` 경로의 에셋을 사용합니다.
- 사운드: `merged/sound/` 경로의 오디오 파일을 사용합니다.
- 에임 트레이너 총소리: `merged/sound/gunshot.mp3`를 사용합니다. (없어도 게임 진행 가능)

## 참고
- `merged/` 가 통합 허브 역할을 하며, 반드시 이 폴더를 서버 루트로 띄워야 Runner의 이미지 경로(`/image/...`)가 올바르게 로드됩니다.
- 첫 클릭 시 에임 트레이너는 Pointer Lock과 오디오가 활성화됩니다(브라우저 정책).
