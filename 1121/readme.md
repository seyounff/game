# Unity 코인 게임 실습 정리

## 📌 개요

이 프로젝트는 Unity를 활용하여 기본적인 게임 제작 과정을 실습한 내용을 정리한 것입니다. 에셋 임포트부터 스테이지 구성, 메테리얼 적용, 장애물 배치, UI 텍스트 표시, 충돌을 활용한 코인 게임 제작까지 단계적으로 연습했습니다.

---

## 📁 1. 에셋(Asset) 임포트 및 스테이지 생성

* Unity Asset Store 또는 로컬 파일에서 필요한 모델·텍스처·프리팹 등을 임포트함
* Plane, Cube 등을 배치하여 간단한 스테이지 구성
* Player 오브젝트 생성 후 Rigidbody 추가

---

## 🎨 2. 메테리얼(Material) 적용

* 새 Material 생성 후 색상 또는 Texture 적용
* 스테이지 및 오브젝트에 메테리얼을 적용해 시각적 요소 강화
* Shader / Rendering Mode 조정으로 표현 방식 실습

---

## 🚧 3. 장애물 배치

* Cube 등 기본 오브젝트를 활용해 장애물 배치
* Transform(위치·회전·크기)을 조정하여 다양하게 구성
* 스크립트를 통한 장애물 움직임 테스트(좌우 이동 등)

---

## 📝 4. 화면 텍스트(UI Text) 실습

* Canvas 생성 후 Text 또는 TextMeshPro 오브젝트 추가
* 점수 표시, 시간 표시 등 UI 구성 실습
* 스크립트에서 UI 업데이트 코드 테스트

---

## 💰 5. 물체 충돌을 이용한 코인 게임 구성

* Coin 오브젝트 생성 및 회전 애니메이션 추가
* Player와 Coin 간 충돌 감지(OnTriggerEnter 사용)
* 충돌 시 코인 제거 + 점수 증가
* UI Text와 연동하여 실시간 점수 표시 구현
* 기본적인 게임 루프 설정

---

## ✔️ 배운 점

* Unity 에셋 임포트 및 환경 구성
* Material 및 UI 사용법
* 충돌 처리(OnCollisionEnter, OnTriggerEnter)
* 간단한 코인 획득 게임 로직 구현

---

## 📚 앞으로의 목표

* 점프/이동 등 플레이어 조작 고도화
* 다양한 장애물 패턴 추가
* 게임 시작/종료 UI 구성
* 사운드 추가 및 애니메이션 고도화
