# 🎮 C언어 콘솔 게임 프로그래밍 기초 (09/19)

## 📌 개요

C 언어를 사용하여 콘솔 환경에서 숫자를 처리하고 도형을 제어하는 기능을 구현하는 방법을 학습했습니다.

---

## 📝 주요 학습 내용

### 1. 숫자의 자릿수 분리 및 표현

입력된 정수를 각 자릿수(단위)별로 분리하여 출력하는 다양한 방법을 학습했습니다.

#### 1.1 문자열로 처리하는 방법

* **높은 단위부터 출력**: 문자열의 첫 번째 인덱스부터 순서대로 출력
* **낮은 단위부터 출력**: 문자열의 마지막 인덱스부터 역순으로 출력

#### 1.2 정수형으로 처리하는 방법

| 방법 | 구현 원리 | 사용 함수 |
|------|----------|----------|
| **높은 단위부터** | `log10`으로 자릿수 계산, `pow(10, i-1)`로 나누면서 몫 출력 | `log10()`, `pow()` |
| **낮은 단위부터** | 10으로 나눈 나머지(`% 10`) 출력, 숫자를 10으로 나눔 (`/= 10`) | `while` 반복문 |

#### 1.3 재귀적 호출을 이용한 자릿수 출력

* **높은 단위부터**: 재귀 호출 먼저 → 반환 시 출력
* **낮은 단위부터**: 출력 먼저 → 재귀 호출

### 2. 디지털 숫자 출력

정수를 **5x4 형태** 배열 패턴으로 디지털 숫자 형태 출력

* 숫자 패턴: 0~9 각 숫자를 1차원 배열에 저장 (1은 "■", 0은 공백)
* 재귀 호출로 각 자릿수 분리 및 출력

### 3. 콘솔 도형 제어

3x3 행렬 블록을 화살표 키로 이동, 스페이스 키로 회전시키는 게임 로직

| 함수 | 기능 |
|------|------|
| `gotoxy(x, y)` | 커서를 원하는 좌표로 이동 |
| `print_shape()` | 3x3 행렬 블록 모양 출력 |
| `rotation_right()` | 블록을 시계 방향 90도 회전 |
| `move_shape()` | 블록 이동 및 화면 경계 검사 |
| `move_control()` | 키 입력 처리 (화살표, 스페이스) |
