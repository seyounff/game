### 4. 난수 생성 (Random Number Generation)

#### 4.1 범위 내 난수 생성 방법

특정 범위의 정수를 생성하기 위해 `rand()` 함수와 나머지 연산자 `%`를 조합하여 사용합니다.

```c
#include <stdlib.h>
#include <time.h>

int main() {
    srand(time(NULL));  // 시드 초기화
    
    int num = rand() % 45 + 1;  // 1 ~ 45 범위의 난수
    int num2 = rand() % 100;    // 0 ~ 99 범위의 난수
    
    return 0;
}
```

**공식:**
* `rand() % n` → 0 ~ n-1 범위
* `rand() % n + m` → m ~ m+n-1 범위

### 5. 가변 인수 (Variable Argument)

#### 5.1 개념

함수 호출 시 **인자의 개수가 고정되지 않고 달라질 수 있는 경우**를 처리하는 방법입니다.

#### 5.2 대표적인 예

* `printf()` : 출력할 변수 개수에 따라 인자가 다름
* `scanf()` : 입력 변수 개수에 따라 인자가 다름

#### 5.3 원형 정의 규칙

```c
#include <stdarg.h>

// 반드시 고정 매개변수 1개 이상 필요
// 이후에 , ... (ellipsis) 사용
int example_function(int fixed, ...);
```

**사용 단계:**
1. `va_list` 타입 변수 선언
2. `va_start()`로 초기화
3. `va_arg()`로 인자 접근
4. `va_end()`로 정리
